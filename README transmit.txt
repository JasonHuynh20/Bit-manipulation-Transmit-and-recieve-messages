Program author: Jason Huynh

Purpose: Create a program that accepts user input as a char, turns the input into shorts and set parity bits for error correction

Source files: bit_manipulation.c and bit_manipulation.h

exact compilation command: "gcc transmit.c bit_manipulation.c"

Launching and operation instructions:
1. To launch the program, compile the files then type "./a.out"
2. When you first launch the program, it will prompt you to enter the message you want to transmit.
3. Input the message then click enter.
The program wil then return the integers that represent your message.
